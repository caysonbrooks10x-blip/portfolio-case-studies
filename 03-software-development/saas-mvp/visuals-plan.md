# FieldSync Case Study — Visuals Plan

## Hero Images (3)

### 1. Dispatch Board Dashboard
- **Concept:** Screenshot of the live dispatch board with 8+ jobs visible
- **Style:** Light mode, clear typography, subtle shadows
- **Copy overlay:** "Real-time dispatch for field service teams"
- **Format:** PNG, 1920x1080 (16:9)
- **Usage:** Website hero, proposal cover, social media

### 2. Mobile PWA in Action
- **Concept:** Field tech using app on iPad at job site (stock photo, licensed)
- **Style:** Bright, professional, shows outdoor/industrial context
- **Copy overlay:** "Voice notes. Photo capture. Offline sync."
- **Format:** PNG, 1200x1600 (3:4)
- **Usage:** Portfolio slides, case study headers

### 3. Before/After Dashboard
- **Concept:** Split screen — legacy software UI vs FieldSync dashboard
- **Style:** Annotated callouts highlighting improvements
- **Copy overlay:** "Built for how contractors actually work"
- **Format:** PNG, 1920x1080 (16:9)
- **Usage:** Proposal comparisons, sales decks

## Architecture Diagrams (2)

### 1. System Architecture
- **Concept:** C4 model diagram showing frontend → API → database → integrations
- **Style:** Clean lines, brand colors, consistent iconography
- **Tools:** Figma, exported as SVG
- **Format:** SVG + PNG at 1920x1080
- **Usage:** Technical documentation, architecture reviews

### 2. Data Flow Diagram
- **Concept:** How a job moves through the system (create → dispatch → complete → invoice → payment)
- **Style:** Linear flow with decision nodes
- **Format:** SVG + PNG
- **Usage:** Onboarding materials, developer docs
