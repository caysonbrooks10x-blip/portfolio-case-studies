# Nordic Table — Case Study Summary

## The Brief
Nordic Table needed three custom e-commerce features: room-by-room galleries, visual search, and WebAR product visualization — integrated with their Shopify store in 10 weeks.

## Approach
- 2-week discovery with analytics analysis and customer interviews
- Custom Shopify storefront (React + Hydrogen)
- TensorFlow.js visual similarity matching
- WebAR via model-viewer (no app download)
- Performance-first: 1.8s load on 3G
- A/B testing to validate every assumption

## Results
- **Session duration:** +78%
- **Pages per session:** +113%
- **Add-to-cart rate:** +44%
- **Checkout conversion:** +10%
- **Overall conversion:** +39%
- **Revenue attributed:** $1.8M (3 months)
- **AOV increase:** $42

## Tech Stack
React 18, TypeScript, Node.js, TensorFlow.js, Shopify Storefront API, model-viewer, Cloudflare, Tailwind CSS

## Key Differentiator
Performance-first approach: obsessed over 3G load times while competitors' AR took 8+ seconds. 89% of users who saw AR actually used it — because it loaded instantly.
