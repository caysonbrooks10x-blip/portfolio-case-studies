# Prism Health — Case Study Summary

## The Brief
Prism Health needed a workflow automation dashboard for care coordinators — unified patient data from 14 EHRs, automated care plans, multi-channel outreach, and real-time quality metrics. HIPAA compliant.

## Approach
- 3-week embedded discovery with care coordination teams
- FHIR R4 integration layer for EHR interoperability
- Visual workflow automation engine
- Automated care plan generation with evidence-based guidelines
- Multi-channel outreach (SMS, email, voice)
- Phased rollout by hospital network

## Results
- **Manual work:** 4+ hours → 45 min/day (81% reduction)
- **Care plan time:** 45 min → 8 min (82% faster)
- **Care plan completeness:** 77% → 98%
- **Outreach volume:** 7x increase
- **No-show rate:** 23% → 11% (52% reduction)
- **Compliance findings:** 6/year → 1/year
- **Year 1 Value:** $11.3M

## Tech Stack
React 18, TypeScript, Node.js, PostgreSQL, Elasticsearch, Redis, FHIR R4, Twilio, SendGrid, Auth0, Bull

## Key Differentiator
Automation that respects clinical judgment — automate pattern-matching (who needs outreach), leave clinical decisions to nurses. This was the key to clinician adoption.
