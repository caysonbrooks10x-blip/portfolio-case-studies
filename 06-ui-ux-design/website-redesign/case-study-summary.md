# Meridian Health — Website Redesign Summary

**Client:** Meridian Health | **Project:** High-Conversion Website Redesign | **Timeline:** 12 Weeks

## The Problem
Telehealth platform losing patients to inferior competitors with better websites. Conversion at 2.1%, mobile at 0.8%.

## The Solution
Mobile-first redesign with frictionless booking, trust-first design, and performance optimization.

## Key Results
- Conversion: 2.1% → 6.8%
- Mobile conversion: 0.8% → 4.2%
- Bounce rate: 78% → 42%
- Booking abandonment: 67% → 18%
- PageSpeed: 34 → 89

## My Role
Lead UX/UI Designer with 4-person team.

## Deliverables
- 12 redesigned pages
- Complete design system
- Mobile-first booking flow
- A/B tested and validated
- WordPress headless implementation

## Tools
Figma, Sketch, InVision, Hotjar, Next.js, Tailwind CSS
