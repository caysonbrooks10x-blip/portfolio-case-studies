# Prism Analytics — Dashboard Redesign Case Study

**Redesigning a data analytics platform to unlock user potential.**

---

## The Challenge

Prism Analytics had a world-class analytics engine users couldn't figure out how to use. Their task completion rate of 34% was costing them renewals and deals.

## Our Approach

We conducted deep user research across 24 participants, analyzed 50,000 user sessions, and completely reimagined the dashboard experience with a focus on progressive disclosure and role-based defaults.

## The Solution

A complete UX overhaul featuring:
- **Command bar** for power users (4x fewer clicks)
- **Progressive filter panel** (340% increase in filter usage)
- **Role-based smart defaults** (83% reduction in time-to-first-insight)
- **Insight Card system** (560% increase in customization)

## Results

- Task completion: **34% → 78%**
- Time-to-first-insight: **47 min → 8 min**
- NPS: **28 → 61**
- Enterprise renewal: **71% → 94%**
- Support tickets: **-80%**

---

**Tools Used:** Figma, Storybook, Principle, UserTesting, Hotjar
