## Answers to Common Questions

**"How did you balance fun/gamification with the seriousness of finance?"**
We tested extensively. The key was making celebration moments feel earned, not patronizing. Milestones needed to be meaningful achievements, not arbitrary badges.

**"Did dark mode cause accessibility issues?"**
We implemented accessibility features early: high contrast text, configurable color options, reduced motion support. Dark mode tested well with screen readers too.

**"How did you handle the technical complexity of animations?"**
We created Lottie specs for all animations. The animations are vector-based, performant, and platform-independent. Engineering could implement without design involvement for simple variations.

**"What about iOS and Android differences?"**
We created platform-specific adaptations. iOS got SF Symbols and native feel. Android got Material You integration. Both felt native while maintaining brand consistency.
