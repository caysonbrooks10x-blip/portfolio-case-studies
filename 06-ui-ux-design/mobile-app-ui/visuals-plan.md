## Technical Specifications

### Color Palette for Visuals
- Background: #0D0D0F
- Surface: #1A1A1F
- Primary: #6366F1
- Accent: #22D3EE
- Success: #10B981
- Warning: #FBBF24
- Error: #F43F5E

### Typography
- Headlines: Satoshi Bold
- Body: Inter Regular
- Numbers: JetBrains Mono

### Export Settings
- Images: PNG, quality 90%
- Animations: Lottie JSON, GIF fallback
- Video: MP4, H.264

---

## Animation Specifications

### Celebration Animations
- Format: Lottie JSON
- Duration: 2-4 seconds
- Loop: None (plays once)
- File size: <50KB each

### Micro-interactions
- Duration: 150-300ms
- Easing: spring(1, 80, 10)
- Format: Lottie or code
