# Verve — Mobile App UI System Summary

**Client:** Verve | **Project:** Mobile App UI System | **Timeline:** 16 Weeks

## The Problem
Personal finance app losing 68% of new users before completing onboarding. Confusing navigation, "aggressively boring" aesthetic.

## The Solution
Complete UI overhaul with dark mode design, gamification system, and progressive feature discovery.

## Key Results
- D1 Retention: 41% → 67%
- D7 Retention: 18% → 44%
- Onboarding: 32% → 74%
- Feature discovery: 12% → 56%
- Subscription revenue: $42K → $168K/mo

## My Role
Lead Product Designer with 6-person team.

## Deliverables
- 85-component design system
- Dark mode visual language
- Gamification/celebration system
- Complete notification architecture
- 5-screen onboarding flow

## Tools
Figma, Framer, Lottie, Principle, Mixpanel, Amplitude
