# Unified Executive Dashboard for Private Equity

## The Challenge

Meridian Capital Partners, a $2.4B private equity firm, was spending 14+ hours every week manually compiling portfolio data from 7 disconnected systems. Board meetings were preceded by frantic data scrambles, and reconciliation errors were a quarterly reality.

## The Solution

I built a unified reporting ecosystem using Looker and Snowflake that consolidated all data sources into a single source of truth, with 47 standardized KPIs and automated weekly distribution.

## Results

**96% Reduction in Reporting Time**
Weekly reporting: 14 hours → 45 minutes

**4x Faster Board Preparation**
Board packages: 3 days → 18 hours

**Zero Reconciliation Errors**
From 12 errors per quarter to none

**$127K Annual Savings**
Recovered analyst hours alone

**4.2 Month Payback**
On a $185K investment

## Technical Approach

- Snowflake data warehouse with automated ELT
- dbt transformation layer with metric standardization
- Looker semantic layer for consistent definitions
- Row-level security for sensitive fund data
- Mobile-optimized for boardroom iPad use
