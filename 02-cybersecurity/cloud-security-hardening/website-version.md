# Cloud Security Hardening

## Meridian Health — AWS Infrastructure Transformation

---

### The Challenge

Meridian Health, a regional healthcare network serving 2.3 million patients across 12 hospitals, needed to secure their AWS cloud environment following a security incident. With 847 resources across 5 accounts and HIPAA compliance requirements, they needed expert help fast.

### What We Did

Conducted a 10-week comprehensive cloud security engagement:
- Incident forensics and response
- Multi-account CSPM assessment
- Zero-trust network architecture redesign
- HashiCorp Vault deployment
- HIPAA compliance automation

### The Results

- **156 security misconfigurations** identified (4 critical)
- **94% reduction** in attack surface
- **HIPAA audit passed** with zero critical findings
- **$180,000 annual** cyber insurance savings
- **18 months incident-free** post-engagement

### Investment

**$125,000** | **10 weeks** | **3-person team**
