## Northridge Partners — Pre-Launch Quality Checklist

## Messaging Quality Review

### Accuracy
- [x] All positioning claims substantiated by client examples
- [x] Client case studies approved for use (signed releases)
- [x] No competitor references without legal review
- [x] Statistics ("14 years," "73% retention") verified
- [x] Founder credentials verified (McKinsey, Deloitte experience)
- [x] Industry claims accurate (mid-market manufacturing specifics)

### Brand Voice
- [x] Tone: Serious, confident, approachable
- [x] "You/your" perspective consistently
- [x] Active voice throughout
- [x] No hype language ("game-changing," "revolutionary")
- [x] Specificity over vagueness ("when the merger goes sideways" not "during transitions")
- [x] Relationship words over process words ("continuity," "presence," "accountability")

### Positioning
- [x] Core brand statement differentiated from competitors
- [x] Tagline tested for memorability
- [x] Service page messaging aligned with homepage positioning
- [x] LinkedIn content consistent with brand positioning
- [x] Sales collateral reflects new positioning

### Consistency
- [x] Homepage and service pages consistent
- [x] LinkedIn profiles aligned with website positioning
- [x] Capability deck reflects website messaging
- [x] Proposal template uses new positioning language
- [x] No contradictory messaging across touchpoints
