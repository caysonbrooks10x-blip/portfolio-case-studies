# Deliverables: Ad Copy System
## NovaTech — Exact Outputs Provided

---

## Phase 1: Discovery & Strategy

### 1. Search Query Analysis Report
**File:** `novatech-search-query-analysis.xlsx`

- 6 months of search query data
- Categorized by intent (problem-aware, comparison, brand, generic)
- Waste analysis (% of spend on non-converting queries)
- Keyword opportunities identified
- Negative keyword list (500+ terms)

### 2. Competitor Ad Analysis
**File:** `novatech-competitor-ad-analysis.pdf`

- 8 competitors analyzed
- Messaging themes, offers, and differentiation
- Gaps and opportunities for NovaTech
- CTR benchmarks by competitor

### 3. Customer Interview Synthesis
**File:** `novatech-customer-interviews.docx`

- 8 customer interviews (transcripts available)
- Key pain themes extracted
- Quote library organized by pain point
- Language patterns to mirror in copy

### 4. Ad Group Architecture
**File:** `novatech-ad-group-structure.pdf`

- 12 ad groups mapped to keyword themes
- Landing page assignment per ad group
- Audience segmentation recommendations
- Bid strategy by segment

## Phase 2: Ad Copy Development

### 5. Google Ads Copy Document
**File:** `novatech-google-ads-copy.docx`

- 12 ad groups × 3 headlines + 2 descriptions = 36 primary ads
- All extensions configured (sitelinks, callouts, structured snippets)
- A/B variants flagged for testing
- Character count compliance verified

### 6. LinkedIn Ad Copy Document
**File:** `novatech-linkedin-ads-copy.docx`

- 8 sponsored content ads
- 4 InMail templates
- Audience targeting recommendations by ad
- Frequency/capping guidelines

### 7. Landing Page Copy Library
**File:** `novatech-landing-pages.zip`

- 6 dedicated landing pages:
  - /engineering-sprint-planning
  - /vs-jira
  - /stop-missing-deadlines
  - /customer-stories
  - /free-trial
  - /why-engineers-switch
- Each with headline, subheadline, body, CTA, trust elements
- Mobile-optimized variants included

## Phase 3: System & Templates

### 8. Ad Copy Template Library
**File:** `novatech-copy-templates.notion`

Notion database with 40+ templates organized by:
- Funnel stage (awareness, consideration, decision)
- Audience segment (by company size, industry, role)
- Offer type (trial, demo, consultation)
- Problem theme (velocity, blockers, planning chaos)

Each template includes:
- Psychological framing rationale
- Fill-in-the-blank elements
- Example completed copy
- CTR benchmarks where available

### 9. Testing Framework Document
**File:** `novatech-testing-framework.pdf`

- Weekly testing protocol
- Success metrics by ad type
- Statistical significance thresholds
- Decision framework for scaling vs. killing
- Optimization cadence

### 10. Performance Dashboard
**File:** `novatech-performance-dashboard.xlsx`

Google Sheets dashboard with:
- Daily metrics (spend, impressions, clicks, conversions)
- Weekly trend analysis
- A/B test tracker
- ROAS by ad group
- CPL by audience segment
