# Visuals Plan: Ad Copy System
## NovaTech — Creative Alignment Recommendations

---

## Creative Strategy Overview

The visual system should reinforce the message: NovaTech is professional, engineering-focused, and grounded in real data — not abstract concepts.

**Core visual principles:**
- Clean, minimal, technical aesthetic
- Screenshots and real UI over stock photography
- Data visualizations as social proof
- No generic "business meeting" imagery

---

## Image Guidelines by Ad Type

### Google Search Ads
- Primary visual: Text + brand logo only
- No image component in standard search
- Ad extensions carry the visual weight (sitelinks, callouts)
- Brand colors: Navy (#1a2744), White, Gold accent (#c9a227)

### LinkedIn Feed Ads
- **Recommended:** 1200×627px (optimal)
- **Avoid:** Stock photos, abstract imagery, people in suits
- **Use:** Product screenshots with annotation, data visualizations, simple graphic cards with key stats

### LinkedIn Carousel Ads
- 1080×1080px per card
- Maximum 10 cards
- Simple text overlays on clean backgrounds
- Consistent branding across all cards

### Display/Banner Ads
- Standard IAB sizes:
  - 728×90 (leaderboard)
  - 300×250 (medium rectangle)
  - 160×600 (wide skyscraper)
  - 320×50 (mobile leaderboard)
- Minimal text, logo-forward
- Clear CTA button
- High contrast for readability

## Video Concepts (LinkedIn + YouTube)

### Concept 1: Sprint Board Walkthrough (15 sec)
- Screen recording of NovaTech sprint board
- Show velocity tracking, blocker alerts, dependency mapping
- Voiceover: Key stats ("87% sprint commitment accuracy")
- End card with CTA

### Concept 2: Problem/Solution Interview (30 sec)
- Engineering leader on camera discussing their sprint problems
- Transition to NovaTech solution
- Testimonial style, authentic, not polished

### Concept 3: "Setup in 20 Minutes" Demo (20 sec)
- Timer on screen
- Step-by-step: Create workspace → Import projects → Connect GitHub → Run first sprint

## Landing Page Visual Requirements

### Hero Images
- Desktop: 1920×600px, high quality
- Mobile: 800×600px minimum
- WebP format, < 200KB
- No text overlays on hero images (keep clean)

### UI Screenshots
- Use actual NovaTech UI
- Light background preferred
- Annotated versions for feature highlights
- Consistent frame/border style

### Client Photos (Testimonials)
- Real headshots (not illustrations)
- Professional but approachable
- Consistent sizing: 200×200px circular
- Company logo + title below

## Typography for Display Ads

**Primary:** Inter (Google Font)
- Bold for headlines
- Regular for body
- Numerals should be prominent

**Size Guidelines:**
- Headlines: 24-36px (readable at small sizes)
- Body: 14-18px
- CTAs: 16-20px, bold

## Brand Consistency Checklist

- [ ] Logo appears in same position across all ad formats
- [ ] Color palette consistent
- [ ] Typography uses only brand fonts
- [ ] CTA button style consistent (rounded corners, same color)
- [ ] Tagline or brand positioning line consistent
- [ ] No stock photography — real UI and data only

---

## Common Mistakes to Avoid

1. **Stock photography** — Instantly signals "generic B2B." Engineers respond to real UI.
2. **Too much text** — Google and LinkedIn penalize low image/text ratios.
3. **Inconsistent branding** — Different ad agencies or internal teams creating different looks.
4. **Small CTAs** — Mobile ad CTAs need to be minimum 44px touch target.
5. **No contrast** — Ad fatigue is real. High contrast = stand-out.
