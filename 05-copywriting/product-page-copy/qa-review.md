# QA Review: Product Page Conversion Copy
## Lumina Skincare — Pre-Launch Quality Checklist

---

## Copy Quality Review

### Accuracy
- [x] All statistics verified against clinical studies
- [x] Ingredient lists match actual product formulations
- [x] Product names match Shopify catalog exactly
- [x] Pricing matches current store pricing
- [x] Product sizes (oz/ml) accurate
- [x] "Dermatologist formulated" claim substantiated (founder credentials)
- [x] No competitor references without legal review

### FTC Compliance
- [x] "89% reported visible brightening" — sourced from clinical study (n=120)
- [x] "Individual results may vary" where appropriate
- [x] No "typical results" language for unsubstantiated claims
- [x] Before/after photos have disclosure ("Individual results may vary")
- [x] Review quotes verified as from real customers (Verified Buyer badge)

### Brand Voice
- [x] Tone: Premium but accessible, clinical but warm
- [x] "You/your" perspective consistently
- [x] Active voice throughout
- [x] No hype language ("miracle," "revolutionary")
- [x] Specificity over vagueness ("14 days" not "a couple weeks")

### Readability
- [x] Flesch-Kincaid Grade Level: 8-10 (accessible)
- [x] No paragraph > 30 words
- [x] Bullet points for easy scanning
- [x] Key benefits highlighted (bold, not just listed)
