# QA Review: Landing Page Rewrite
## Meridian Consulting — Pre-Launch Quality Checklist

---

## Copy Quality Review

### Accuracy
- [x] All client-reported statistics verified against source data
- [x] Testimonial quotes reviewed and approved by clients (documentation on file)
- [x] No exaggerated or unsubstantiated claims
- [x] Competitor references removed (legal review complete)
- [x] Pricing ($12,000/year) matches current offer documentation

### Brand Voice
- [x] Tone: Professional but direct — not corporate-speak, not too casual
- [x] Sentences are active voice throughout
- [x] No jargon or acronyms that wouldn't be understood by target audience
- [x] Consistent use of "you/your" vs "we/our" — prospects need to feel addressed
- [x] No passive constructions like "can be achieved" or "may be realized"

### Readability
- [x] Flesch-Kincaid Grade Level: 10-12 (appropriate for executive audience)
- [x] Average sentence length: 15-20 words
- [x] No paragraph exceeds 40 words
- [x] Section headers are benefit-oriented, not feature-oriented
- [x] Bullets use parallel structure throughout

### Conversion Elements
- [x] Primary CTA appears above the fold on mobile
- [x] CTA is action-oriented ("Book" not "Learn More")
- [x] No more than 3 CTAs on the page
- [x] Social proof appears before pricing
- [x] Risk reversal language included ("free strategy session")
- [x] Scarcity element included ("20 spots per year")

## Technical Review

### On-Page SEO
- [x] Meta title includes primary keyword and differentiator
- [x] Meta description is compelling and includes CTA
- [x] H1 appears only once
- [x] H2-H4 hierarchy is logical and sequential
- [x] Alt tags on all images
- [x] Internal linking strategy implemented

### Tracking & Analytics
- [x] Google Analytics 4 installed and event tracking configured
- [x] Google Ads conversion tracking verified
- [x] Hotjar tracking code installed
- [x] All form submissions tracked as conversions
- [x] Click-to-call tracked on phone numbers

### Mobile Experience
- [x] Page renders correctly on iOS Safari (latest 2 versions)
- [x] Page renders correctly on Chrome Mobile (latest 2 versions)
- [x] CTA button is minimum 44px touch target
- [x] Forms are mobile-optimized (no tiny checkboxes)
- [x] Fixed bottom CTA bar functions on scroll
- [x] No horizontal scroll on any viewport

### Performance
- [x] Page load < 3 seconds on mobile (tested via PageSpeed Insights)
- [x] Largest Contentful Paint < 2.5s
- [x] Hero image is WebP format, < 200KB
- [x] No render-blocking JavaScript
- [x] Fonts loaded via CDN with display:swap
