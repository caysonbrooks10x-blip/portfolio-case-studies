## A/B Test Results

### Hook Variant Test (Week 1)

| Variant | Allocated Traffic | Completion Rate | Application Rate |
|---------|------------------|-----------------|------------------|
| Version A: Problem-led | 40% | 71% | 4.8% |
| Version B: Story-led | 35% | 65% | 4.1% |
| Version C: Statistic-led | 25% | 62% | 3.4% |

**Winner:** Version A (Problem-led) — deployed as control

### CTA Variant Test (Week 2)

| Variant | Completion Rate | Application Rate |
|---------|------------------|------------------|
| Version A: Apply | 68% | 4.3% |
| Version B: Schedule Call | 68% | 3.1% |

**Winner:** Version A (Apply) — lower friction, higher conversion
