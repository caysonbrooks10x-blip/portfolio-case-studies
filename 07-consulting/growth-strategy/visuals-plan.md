## Animation/Motion Guidelines

### Website Hero Animation
- Metrics count up from 0 on scroll into view
- Duration: 2 seconds total
- Easing: ease-out

### Dashboard Reveal
- Cards fade in sequentially (100ms delay each)
- Numbers animate to final value
- Subtle pulse on highest-performing metric

### Slide Transitions (Presentation)
- Fade transition between slides
- No distracting animations
- Charts animate on click

## Brand Consistency Guidelines

### Color Palette
- Primary: #1a1a2e (dark navy)
- Secondary: #00d4aa (teal/mint)
- Accent: #6366f1 (indigo)
- Success: #22c55e (green)
- Warning: #f59e0b (amber)
- Background Light: #f8fafc
- Background Dark: #0f172a
- Text Primary: #1e293b
- Text Secondary: #64748b

### Typography
- Headlines: Inter Bold / Poppins Bold
- Body: Inter Regular / Open Sans
- Data/Numbers: JetBrains Mono / Roboto Mono
- Font sizes: 48px (hero), 32px (H1), 24px (H2), 18px (body), 14px (caption)

### Spacing
- Section padding: 80px vertical
- Card padding: 24px
- Grid gap: 24px
- Border radius: 12px (cards), 8px (buttons)

## Implementation Notes

1. **Priority:** Dashboard visual (#1) should be completed first as it's the hero asset
2. **Review Cycle:** All visuals should be reviewed by client before publication
3. **Accessibility:** Include alt-text descriptions for all visual assets
4. **Export Formats:** Provide both PNG and SVG where possible
5. **Source Files:** Maintain editable source files (Figma, Sketch, or PowerPoint)
